# DB Shop. A DB assignment.

## 简介

ZJGSU 数据库课程设计.

## 如何运行 ?

安装如下依赖包

**Requirements.txt**

```
pymssql==2.1.5
PyQt5==5.15.6
qt_material==2.8.19
```

1. 执行 `git clone https://github.com/chenghuaWang/DBShop-DB-assignment.git`
2. 执行 `pip install -r requirements`
3. 执行 `python src/main.py`
